Unidad 2:CSS

¿Qué es CSS?

Es el leguaje que aplica estilo a los elementos HTML.

¿Por qué usamos CSS?

Lo usamos por que es mas facil de manijar.

¿Dónde se pone el CSS?

Dentro de una etiqueta de style y el encabezado.

¿Qué etiquetas de estilo ya hemos aprendido? Define qué cambia cada propiedad.

  • Background-color: Se utiliza para para poner color de fondo
  • Color:Se utiliza para poner color al texto.
  • Font-family:Se utiliza para poner fuentes.
  • Text align:Se utiliza para dar alineacion de texto.
  • Font-size:Se utiliza para dar tamano al texto.

Escribe un ejemplo de CSS que ponga el fondo de todas las "p" de color rojo:

p{
color:red;
}
