¿Cómo se crea una carpeta?

Darle clic derecho y despues se le da crear archivo.

¿Cómo se abre una carpeta en VS?

En la parte superior hay signo de crear caperta se le da hay y se crea.

¿Qué es HTML?

Es el leguaje para dar ihnstrucciones a una computadora sobre como hacer una pagina web.

¿Cómo se crea un archivo de HTML en VS?

En la parte superior hay una carpa que parece una hoja se la hay y se le da a crear.

¿Qué etiquetas de HTML ya conocemos? Define cada etiqueta.

  1. Body: Para el contenido
  2. Head: Para la informacion del documento.
  3. Div:Se utiliza para separar el contenido.
  4. a: Para los enlaces.
  5. Strong:Se utiliza para poner palabras en negritas.
  6. Br: Para poner espacio entre un parrafo y otro.
  7. H1-H6:Para los titulos dentro del contenido.
  8. img:Para poner imagenes al documento.
  9. Em:Pone el texto en cursiva.
  10. Span:Se utiliza para marcar una parte del texto.
  11. P: para poner la informacion acerca del tema elegido.
  12. OL: Se utiliza para realizar listas ordenada.
  13. UL:Se utiliza para hacer listas desordenadas.
  14. Doctype: El que indica el leguaje.
    15. il:Es la etiqueta que se usa para hacer las lista ordenas y desordenadas.
  15. style:Es la etiqueta se usa para dar co,lor al documento.
  16. Button:Se utiliza para regresar a la pagina de donde inicia tu pagina de bienvenida.
