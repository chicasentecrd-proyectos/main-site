Unidad 2: CSS

¿Qué es CSS?

Es el lenguaje que aplica estilo a los elementos html

¿Por qué usamos CSS?

Para enbelleser nuestra pagina

¿Dónde se pone el CSS?

Debajo del elemento title antes de cerrar head

¿Qué etiquetas de estilo ya hemos aprendido? Define qué cambia cada propiedad.

  • Font-family=Cambia el tipo de letra
  • Font-size=Cambia el tamaño de la letra
  • color:pink;"=Cambia el color de la letra
  • Background-color:green;"=Modifica el color del fondo
  • text-aling=alinea el texto para que lo puedas poner en el centro

Escribe un ejemplo de CSS que ponga el fondo de todas las ps de color rojo:

p {
