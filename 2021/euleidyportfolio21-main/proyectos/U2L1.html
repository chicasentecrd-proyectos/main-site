Unidad 2:CSS

¿Qué es CSS?

Es el lenguaje que aplica estilo a los elementos html.

¿Por qué usamos CSS?

  • Para separar contenido de estilo
  • Para que las paginas html esten mas limpias y eficientes

¿Dónde se pone el CSS?

Se escribe dentro de head y abres un elemento style y adentro escribimos

¿Qué etiquetas de estilo ya hemos aprendido?

  • Color: este sirve para cambiar el color de las letras.
  • Background-color: este estilo es para cambiar el color de fondo.
  • Font-family: este sirve para cambiar el tipo de letra de la pagina.
  • Text-align: este sirve para poner el texto o encabezado en diferentes lugares de la pagina como, a la izquierda, derecha y centro.
  • Font-size:este cambia el tamano del texto y encabezado.

Escribe un ejemplo de CSS que ponga el fondo de todas las ps de color rojo:

p {
color:red;
}
