UNIDAD 1: Funciones Básicas y HTML

¿Cómo se crea una carpeta?

Damos Click derecho + nuevo + carpeta.

¿Cómo se abre una carpeta en VS?

En la barra de menu en VS elegimos el archivo le damos a abrir carpeta.

¿Qué es HTML?

Es el lenguaje que da instrucciones a la computadora sobre como hacer paginas Web.

¿Cómo se crea un archivo de HTML en VS?

Le damos click al signo de una hoja y el signo de mas en VS y asi creamos nuestro archivo.

¿Qué etiquetas de HTML ya conocemos? Define cada etiqueta.

  1. h1,h2,h3,h4,h5,h6: Encabezados
  2. P: Parrafos
  3. Body: Cuerpo
  4. Br: Para dejar lineas o espacios
  5. Title: Nombre de la pagina en la pestaña
  6. Div: Para dividir por secciones la paginas
  7. Em: Para poner las palabras u oraciones dobladas o cursivas.
  8. Head: Cabeza de la pagina
  9. Strong: Poner las palabras u oraciones en negritas
  10. Style: Para poner estilo a la pagina
  11. Span: para darle un color en especifo a una palabra
  12. Ol: Para hacer listas ordenadas
  13. Ul: Para hacer listas desordenadas
  14. A: Para proporcionar enlaces a la pagina web
  15. Button: Para dar un boton a la pagina, sea para volver al inicio etc.
  16. html: Para darle a conocer a la pagina que se tiene que hacer c
  17. !DOCTYPE html: Para identicar el archivo como documento html.
  18. Img: Para poner imagenes en la pagina Web
  19. li: Para poner puntos o numeros a las listas, sean ordenadas o desordenadas
