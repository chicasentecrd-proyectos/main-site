Unidad 2: CSS

¿Qué es CSS?

Es el lenguaje que aplica estilo a los elementos html

¿Por qué usamos CSS?

para ordenar las instrucciones referentes a la apariencia de un sitio y presentar los contenidos de una página de forma atractiva.

¿Dónde se pone el CSS?

En la etiqueta style

¿Qué etiquetas de estilo ya hemos aprendido? Define qué cambia cada propiedad.

  • Color de fondo: background-Color
  • Fuente: Font-Family
  • Color de texto: color
  • Tamaño del texto: Font-size
  • Alineación de texto: Text-alighn
