¿Cómo se crea una carpeta en VS?

Dar clik derecho; darle a "NUEVA CARPETA" o dar clik dereco al boton con una carpeta.

¿Cómo se crea un archivo de HTML en VS?

click derecho darle a "NUEVA CARPETA" o dar click al boton con una carpeta y un + .

¿Qué es HTML?

Es el lenguaje para dar instrucciones a una computadora de como acer una pagina web

¿Qué etiquetas de HTML ya conocemos? Define cada etiqueta.

  • h1 = El encabezado mas grande
  • p = Es un parrafo
  • h6 = El encabezado mas pequeno
  • div = un contenedor
  • ol = para listas desordenadas
  • li = para listas ordenadas
