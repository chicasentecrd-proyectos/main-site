Unidad 2:CSS

¿Qué es CSS?

Es el lenguaje que aplica estilo a los elementos html

¿Por qué usamos CSS?

para aplicar con mayor facilidad y rapidez estilo a la pagina html

¿Dónde se pone el CSS?

El código se escribe en un archivo . css a parte. Recomendado. El código se escribe en una etiqueta style en el documento HTML.

¿Qué etiquetas de estilo ya hemos aprendido? Define qué cambia cada propiedad.

  • Background color- el cual sirve para cambiarle el color del fondo a una pagina o a una division de la misma
  • color de texto:color- se utiliza para agregarle color al texto.
  • Font- Family es para cambiar la fuente del texto a una de tu preferencia
  • font-size es para cambiar el tamaño de la fuente
  • Text-align se utiliza para poner en el lugar de preferencia tu texto
