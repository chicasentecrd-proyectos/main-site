Unidad 2: CSS

¿Qué es CSS?

Es el lenguaje que aplica estilo a los
elementos html.

¿Por qué usamos CSS?

Lo usamos para que poner el estilo a
las paginas sea mas facil.

¿Dónde se pone el CSS?

El css se escribe en la cabeza de las
paginas, dentro de style .

¿Qué etiquetas de estilo ya hemos aprendido? Define qué cambia cada propiedad.

  • background-color: color de fondo .
  • color: color de texto .
  • font-family: fuentes .
  • font-size: tamaÑo de texto .
  • tex-align: alineacion .
