Unidad 2:Ccs

a:¿Qué es CSS?

El CCS o Combined Charging System hace referencia al sistema enchufable de carga combinada.
También se le conoce como Sistema de Carga Combinado, CCS Combo o Combo 2.
Este sistema se utiliza para la carga de vehículos eléctricos, independientemente de
que su carga sea alterna (AC) o de corriente continua (DC).

b:¿Por qué usamos CSS?

CSS es uno de los lenguajes más importantes que se utilizan para ordenar las instrucciones
referentes a la apariencia de un sitio y presentar los contenidos de una página
de forma atractiva. De este modo, HTML se emplea para estructurar el
contenido de un sitio, mientras que CSS, para estructurar su presentación

c:¿Dónde se pone el CSS?

Con las hojas de estilo internas, las instrucciones CSS se encuentran dentro del documento HTML.
Así, puedes integrar CSS al principio del archivo HTML o colocarlo en el código fuente.
Con este método, una buena comprensión básica de HTML y su sintaxis es una ventaja
