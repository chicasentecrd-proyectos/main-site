Unidad 2: css

¿que es el css?

es el lenguaje que aplica estilo a los HTML.

¿Por qué usamos CSS?

para hacer el desarollo de grandes sitios web mas facil

¿Dónde se pone el CSS?

en la etiqueta estilo

¿Qué etiquetas de estilo ya hemos aprendido? Define qué cambia cada propiedad.

  • color de fondo: background-color
  • color de texto: color
  • fuentes: font-family
  • tamaño de texto: font-size
  • alineasion del texto: text-aling
