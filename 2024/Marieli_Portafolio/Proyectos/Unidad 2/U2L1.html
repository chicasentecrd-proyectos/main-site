unidad 2: CSS

¿Qué es CSS?

Es la herramienta que aplica estilo a los elementos html

¿Por qué usamos CSS?

para que sea mucho mas facil y con mas estilo el desarrollo de las paginas web

¿Dónde se pone el CSS?

En la etiqueta estilo

¿Qué etiquetas de estilo ya hemos aprendido? Define qué cambia cada propiedad.

  • color de fondo:background/color
  • tipo de letra: font family
  • tamaño de texto:font zise
  • alineacion del texto: text align
  • color del texto:texto
