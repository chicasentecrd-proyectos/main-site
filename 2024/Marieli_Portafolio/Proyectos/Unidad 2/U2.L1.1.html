¿Qué es HTML?

Es un lenguaje marcado de hipertexto

¿Qué etiquetas de HTML ya conocemos? Define cada etiqueta.

  • br= es la etiqueta que no se cierra y sirve para que el programa reconozca que es un salto de linea
  • em= pone el texto en cursiva
  • strong=pone el texto en negrita
  • h1 hasta h6= sirven para encabezado
  • p= este es un parrafo
